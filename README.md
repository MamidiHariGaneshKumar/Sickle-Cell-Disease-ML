# Sickle Cell Disease Detection using Machine Learning

This project detects **Sickle Cell Disease** using microscopic blood cell images and Machine Learning techniques.
The system analyzes blood cell images and predicts whether the cell is **normal or affected by sickle cell disease**.

---

## Features

* Upload blood cell images
* Image preprocessing
* Machine Learning prediction
* Secure data processing using **Homomorphic Encryption**
* Web interface for user interaction

---

## Technologies Used

### Backend

* Python
* Flask

### Machine Learning

* Logistic Regression
* NumPy
* Scikit-learn

### Encryption

* Homomorphic Encryption (CKKS)

### Frontend

* HTML
* CSS
* JavaScript

---

## Project Structure

Sickle-Cell-Disease-ML
│
├── Positive (Sickle cell images dataset)
├── Negative (Normal blood cell images dataset)
├── static (CSS and JavaScript files)
├── templates (HTML pages)
├── app.py (Flask application)
├── model.py (Machine learning model)
├── requirements.txt (Project dependencies)

## How to Run the Project

### 1 Install Dependencies

pip install -r requirements.txt

### 2 Run the Application

python app.py

### 3 Open Browser

http://127.0.0.1:5000/
